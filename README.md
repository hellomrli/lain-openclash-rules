# Lain OpenClash Rules

Lain 标准版规则：为 OpenClash / Mihomo 订阅转换准备的精简规则模板。

## 特点

- 不包含机场订阅链接
- 不包含节点名称、节点地址或节点信息
- 仅保留香港 / 美国 / 日本三个地区节点组
- 移除 YouTube、Netflix、Disney+、HBO、PrimeVideo 等媒体细分策略组；保留 Steam 与 Bahamut
- 增加 AI 服务、DMM、乌龟服、Bilibili 策略组，并保留 Steam、Bahamut 独立组
